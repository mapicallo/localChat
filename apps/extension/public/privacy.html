LocalChat · Browser extension · Effective date: 2 June 2026

LocalChat · Extensión de navegador · Fecha de vigencia: 2 de junio de 2026

This policy describes how the LocalChat extension (“the Extension”) handles information. LocalChat provides a local chat using your browser’s on-device language model (Gemini Nano via Chrome’s Prompt API). Your chat text is not sent to servers operated by the developer for processing.

Single purpose

The Extension has one purpose: a side-panel chat assistant that runs on your device, optionally using visible text from a web page you explicitly attach with Use this page or highlighted text with Use selection, or local files you pick with Attach document / Attach image.

Who we are

LocalChat is part of the AI4Context extension family. See ai4context.com.

What the Extension does

  • Runs prompts through Chrome’s built-in on-device model in the side panel.
  • When you click Use this page, reads visible text from the active tab in the current window (via a one-time script injection you confirm) to answer your questions locally.
  • When you click Use selection, reads only the text you highlighted on the active tab (one-time script injection, no extra dialog) to answer locally.
  • When you click Attach document, opens a file picker; text is extracted from PDF or plain-text files in the extension panel and sent only to the on-device model—nothing is uploaded to AI4Context servers.
  • When you click Attach image, opens a file picker; the image is resized if needed and sent only to the on-device model for multimodal prompts—processed locally.
  • Stores UI language preference and keeps the model session in memory while you chat.

Data we do not collect

No account is required. We do not operate a backend that receives your chat, attached page text, or files you pick. Conversation content is not uploaded to AI4Context servers as part of the core feature.

Local storage

The Extension uses chrome.storage.local for:

  • UI language preference (lc_locale),
  • saved conversations on this device (lc_conversations, up to 40 chats—local only, not synced to AI4Context servers).

On-device model files are managed by Google Chrome, not by this Extension’s server.

Permissions

  • Side panel: chat UI.
  • Storage: language preference.
  • Active tab + scripting + tabs: only when you choose Use this page or Use selection, to read visible or selected text from that tab.

Third parties

Google Chrome downloads and runs Gemini Nano on your device. See Google’s documentation and privacy notices for built-in AI. Websites you attach may have their own policies.

Contact

Questions: github.com/mapicallo/localChat or ai4context.com.

Esta política describe cómo la extensión LocalChat («la Extensión») trata la información. LocalChat ofrece un chat local con el modelo de lenguaje integrado en tu navegador (Gemini Nano mediante la Prompt API de Chrome). El texto del chat no se envía a servidores del desarrollador para procesarlo.

Propósito único

La Extensión tiene un solo propósito: un asistente de chat en panel lateral que funciona en tu dispositivo, opcionalmente usando texto visible de una página web que tú adjuntas con Usar esta página o texto marcado con Usar selección, o archivos locales que eliges con Adjuntar documento / Adjuntar imagen.

Quiénes somos

LocalChat forma parte de la familia de extensiones AI4Context. Consulta ai4context.com.

Qué hace la Extensión

  • Ejecuta prompts con el modelo integrado en Chrome en el panel lateral.
  • Cuando pulsas Usar esta página, lee el texto visible de la pestaña activa en la ventana actual (mediante un script que tú confirmas) para responder en local.
  • Cuando pulsas Usar selección, lee solo el texto que marcaste en la pestaña activa (script puntual, sin diálogo extra) para responder en local.
  • Cuando pulsas Adjuntar documento, abre un selector de archivos; extrae texto de PDF o archivos de texto en el panel de la extensión y lo envía solo al modelo local—nada se sube a servidores de AI4Context.
  • Cuando pulsas Adjuntar imagen, abre un selector; la imagen se redimensiona si hace falta y se envía solo al modelo local para prompts multimodales—procesado en local.
  • Guarda la preferencia de idioma de la interfaz y mantiene la sesión del modelo en memoria mientras chateas.

Datos que no recopilamos

No se requiere cuenta. No operamos un backend que reciba tu chat, texto de página adjunto o archivos que elijas. El contenido de la conversación no se sube a servidores de AI4Context como parte de la función principal.

Almacenamiento local

La Extensión usa chrome.storage.local para:

  • preferencia de idioma (lc_locale),
  • conversaciones guardadas en este dispositivo (lc_conversations, hasta 40 chats—solo local, sin sincronizar con servidores de AI4Context).

Los archivos del modelo en el dispositivo los gestiona Google Chrome, no un servidor de esta Extensión.

Permisos

  • Panel lateral: interfaz de chat.
  • Storage: preferencia de idioma.
  • Pestaña activa + scripting + tabs: solo cuando eliges Usar esta página o Usar selección, para leer texto visible o seleccionado de esa pestaña.

Terceros

Google Chrome descarga y ejecuta Gemini Nano en tu dispositivo. Consulta la documentación y avisos de privacidad de Google sobre IA integrada. Los sitios web que adjuntas tienen sus propias políticas.

Contacto

Preguntas: github.com/mapicallo/localChat o ai4context.com.
